# sistema_triagem_hospitalar_trabalhoIA
Projeto desenvolvido como trabalho final para a disciplina de Inteligência Artificial, cujo objetivo principal é aplicar conceitos de Machine Learning Supervisionado para classificar o nível de urgência de pacientes em Prontos-Socorros, mitigando o risco de erros humanos e otimizando o tempo de resposta em ambientes hospitalares sobrecarregados.
